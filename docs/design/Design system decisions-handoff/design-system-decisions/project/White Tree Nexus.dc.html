10 Guia de aplicação · tokens → PrimeNG

Os tokens --wtn-* são definidos em styles.scss em dois escopos (:root claro e .dark) e mapeados para as CSS variables do PrimeNG.

Token PrimeNGToken WTN
--p-primary-colorvar(--wtn-primary)
--p-content-backgroundvar(--wtn-card)
--p-content-border-colorvar(--wtn-border)
--p-text-colorvar(--wtn-text)
--p-text-muted-colorvar(--wtn-text-2)
--p-border-radiusvar(--wtn-r-md)
--p-focus-ring-colorvar(--wtn-focus)
